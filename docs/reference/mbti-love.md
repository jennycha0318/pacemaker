# MBTI 연애 성향 분석 — 큐핏 의사결정 참고 자료

> **출처:** 박선영, 「MBTI 성격유형에 따른 사랑의 구성요소와 사랑유형분석」(울산대학교 가정관리학과 석사학위논문, 2001).
>
> **이 자료의 성격(반드시 인지):** 한국 대학생 약 215~219명 표본, **2001년 석사논문 1편**에 기반한 참고 자료입니다. 표본 규모·시대가 한정적이고, **논문 저자 스스로 결론에서 "MBTI 기질·기능별 사랑 차이는 대체로 무의미하다"**고 밝혔습니다. 즉 MBTI는 연애 성향을 강하게 예측하는 도구가 아닙니다. 큐핏에서는 **진단 점수·타이밍 로직에 반영하지 않고**, 사용자 이해를 돕는 보조 **'참고 레이어'**로만 활용합니다. 항상 단정("당신은 ~한 사람입니다")이 아니라 **경향성("~한 편일 수 있어요")** 수준으로만 인용하세요.
>
> 용어(사랑의 3요소·6가지 사랑유형) 정의는 맨 아래 [부록]에 있습니다.

## 1. MBTI 지표별 연애 성향 경향 〈핵심〉

> **해석 주의:** 논문 본문 표 분석에서는 "4개 지표 단독으로는 사랑 구성요소에 유의한 차이가 없다"고 했으나, 영문초록·부록 상관표에서는 E/I·J/P 일부에서 유의한 경향이 보고됩니다. 아래는 그중 비교적 반복적으로 확인된 **방향만** 정리한 것이며 모두 '경향' 수준입니다. **구체 수치는 표마다 판독값이 엇갈리므로 인용하지 말고 방향성만** 사용하세요.

### E / I — 외향 · 내향
- **구성요소:** 외향(E)이 내향(I)보다 친밀감·열정·헌신이 다소 높게 나오는 경향(부록 상관표에서 E와 약한 +상관). 단 본문에선 비유의 처리된 곳도 있어 강한 단정 불가.
- **사랑유형:** 외향(E) = 낭만적(Eros)·유희적(Ludus) 사랑이 높은 경향(적극적·표현적). 내향(I) = 좋은 친구 같은 사랑(Storge)에 가까운 경향.

### S / N — 감각 · 직관
- **구성요소:** 뚜렷한 관계 약함(대부분 비유의). 초록에선 친밀감·헌신에서만 일부 유의하다고 언급되나 방향 불명확.
- **사랑유형:** 감각(S) = 논리적(Pragma) 사랑, 직관(N) = 낭만적(Eros)·이타적(Agape) 사랑 쪽으로 약하게 기우는 경향.

### T / F — 사고 · 감정
- **구성요소:** 유의한 관계 거의 없음.
- **사랑유형:** 감정형(F) = 이타적(Agape)·낭만적(Eros)·우애적(Storge) 사랑이 높은 경향(공감·관계 조화 중시). 사고형(T) = 논리적(Pragma)·유희적(Ludus) 사랑이 상대적으로 높은 경향(객관·거리 두기).

### J / P — 판단 · 인식
- **구성요소:** 뚜렷한 관계 약함(비유의).
- **사랑유형:** 판단형(J) = 헌신적·우애적(Storge)·이타적(Agape) 사랑. 인식형(P) = 유희적(Ludus) 사랑이 높은 경향. 부록 상관표: J/P가 유희적(Ludus)과 -상관, 소유적(Mania)과 +상관(즉 J 쪽이 소유적, P 쪽이 유희적 경향).

### 종합 경향
사랑의 **'양(구성요소)'은 성격보다 관계 단계·공통 경향**에 가깝고, **성격 차이는 '사랑의 방식(유형)'에서 더 드러납니다.** 거칠게 요약하면:
- **F · J · E 계열** → 이타적·우애적·낭만적 사랑(**헌신·관계지향**)
- **T · P 계열** → 논리적·유희적 사랑(**자유·거리 두기**)

### 구성요소 ↔ 사랑유형 상관 (참고)
- 친밀감·열정 → 이타적(Agape)·유희적(Ludus)·낭만적(Eros)과 양(+)상관(초록 기준). 단 본문 표에선 유희적(Ludus)이 친밀감·헌신과 -상관으로도 보고되어 **Ludus 방향은 엇갈림 — 주의**.
- 헌신 → 낭만적(Eros)과 양(+)상관.
- 일관된 메시지: **진지·성숙한 사랑(Eros·Agape)일수록 3요소가 함께 높아지고, 가벼운 사랑(Ludus)일수록 헌신이 낮아지는** 방향.

## 2. 16유형별 연애 특징 〈핵심〉

> 논문 본문의 '유형별 사랑 성향 서술'(정성적 묘사)과 일부 결과표 순위 패턴에서 확인된 범위입니다. 통계적으로 검증된 정밀 수치가 아니라 **서술 수준**이며, 논문이 개별 언급하지 않은 유형은 '데이터 제한'으로 표기합니다.

| 유형 | 논문에서 확인된 연애 성향 (서술 수준) |
|---|---|
| **ISTJ** | 신중·과묵, 책임감이 강하고 한번 정한 관계에 일관되게 헌신. 논리적(Pragma) 사랑 상위로 거론. |
| **ISFJ** | 조용히 상대를 돌보고 안정된 관계를 지키는 헌신적 사랑. 우애적(Storge)·친밀감 높은 편. |
| **INFJ** | 깊고 의미 있는 관계를 추구. 통찰로 상대를 이해하려 하며 헌신적. 이타적(Agape) 사랑 상위 경향. |
| **INTJ** | 독립적·전략적. 연애에도 비전·계획이 뚜렷하고 자기 주관이 강함. 논리적(Pragma) 사랑 상위 거론. |
| **ISTP** | 자유·독립 지향, 즉흥적·실제적이며 감정 표현은 절제하는 편. |
| **ISFP** | 온화·헌신적. 말보다 행동으로 표현하는 조용하고 따뜻한 사랑. |
| **INFP** | *데이터 제한* — 표본 비율이 가장 높았으나(약 9%대) 개별 연애 성향 서술 미확보. |
| **INTP** | 논리·독립적, 감정 표현에 서툴고 사랑을 분석적으로 접근. |
| **ESTP** | 활동적·모험적, 즉흥적이고 현재 중심의 자유로운 사랑. 유희적(Ludus) 사랑 상위 경향. |
| **ESFP** | 사교적·낙천적, 현재의 즐거움과 함께하는 활동 중심 사랑. 낭만적(Eros) 사랑 상위 거론. |
| **ENFP** | 열정적·낭만적, 가능성과 감정 표현이 풍부. 여러 사랑유형에서 높게 거론. |
| **ENTP** | 새로움·지적 자극 추구의 활달한 사랑. 유희적(Ludus) 사랑 상위 경향. |
| **ESTJ** | 책임감·현실적. 약속·계획을 중시하는 안정 지향적 사랑, 자기 방식대로 표현. |
| **ESFJ** | **거의 모든 사랑유형에서 점수가 가장 높게 나온 유형.** 따뜻하고 정 많으며 배려·헌신이 큼. 우애적·낭만적·이타적 사랑 모두 상위. |
| **ENFJ** | 동정심·동료애가 크고 진정성 있게 상대를 아낌. 책임감 강한 이상적·헌신적 사랑. 이타적(Agape) 사랑 상위. |
| **ENTJ** | 주도적·계획적, 목표 지향적이며 자기 비전대로 관계를 이끔. 여러 구성요소에서 높게 거론. |

**큰 흐름(데이터 기반 경향):** NF = 낭만·의미·헌신 / SJ = 책임·안정·헌신 / SP = 자유·즐거움·즉흥 / NT = 논리·독립·전략. **F·E 계열(특히 ESFJ·ENFJ류)이 친밀·헌신·이타적 사랑에서 가장 높았고, EP·TP 계열이 유희적 사랑에서 높은** 경향이었습니다.

**데이터 제한 안내:** 16유형을 균등하게 다룬 통계표가 아니라 일부 유형 중심 서술이라, 유형 간 비교의 신뢰도는 균일하지 않습니다(특히 INFP 개별 서술 미확보).

## 3. 의사결정 분석 활용 가이드

큐핏의 진단(점수·타이밍·행동)은 규칙 엔진이 결정하고, **MBTI는 그 위에 얹는 '해석·문구 톤 조정' 보조 레이어**입니다. 아래는 유형 경향을 의사결정·코칭 톤에 어떻게 참고할지에 대한 가이드입니다(모두 경향 수준, 점수 미반영).

### 활용 원칙
- **보조 레이어로만:** 점수·타이밍 추천 로직에 반영하지 않음. 사용자가 자기 연애 방식을 언어로 이해하게 돕는 '거울'.
- **경향성 어투 고정:** "당신은 헌신형입니다"(X) → "ENFJ 성향이라면 관계에 책임감을 크게 느끼는 편일 수 있어요"(O).
- **3요소·6유형 언어로 번역:** 상황을 친밀감/열정/헌신 또는 6가지 사랑유형 한국어명으로 풀어 설명(예: "지금은 열정보다 친밀감이 앞선 단계일 수 있어요").
- **강점 프레이밍:** 우열이 아니라 스타일 차이로(논리적 사랑 = 신중함, 유희적 사랑 = 자유로움).

### 유형 경향 → 코칭 톤 조정 예시 (참고)
- **F · J · E 계열(헌신·관계지향):** 진정성·안정·직면 권유가 잘 맞음. 진지한 진전(고백·재회) 제안 시 책임감·미래 언급이 설득력 있음.
- **T · P 계열(자유·거리 두기):** 부담 낮은 프레이밍, 구속감 줄이는 접근. "퇴로를 열어두는" 문구가 잘 맞음. 상대의 헌신 기대치를 함께 점검.
- **소유적(Mania) 경향 신호(불안형·J 쪽):** 과한 확인·집착으로 흐를 수 있으니 '확인 줄이기·나 전달법'을 더 강조(기존 위험 경고와 결합).
- **유희적(Ludus) 경향 신호(P 쪽):** 관계 진전 전에 **상대·본인의 헌신 의지**를 가볍게 확인하도록 안내.

### 톤 가이드 (큐핏 디자인 정체성 — 부드럽고 따뜻하게)
- 단정·진단형 어휘("반드시", "당신의 유형은 ~다") 회피, 권유·관찰형("~한 편이에요", "~할 수 있어요") 사용.
- 부정적 라벨은 중립·성장 지향으로 순화: 소유적 → "애정 확인이 중요한", 유희적 → "구속을 덜 좋아하는".

### 한계 (반드시 내부 인지)
- 근거가 2001년 석사논문 1편, 한국 대학생 소표본(약 215~219명)으로 **일반화·최신성 모두 제한적**.
- 논문 저자 자신이 "기질·기능별 사랑 차이는 무의미하다"고 결론지었을 만큼 **MBTI–연애 성향의 통계적 연결은 약함**.
- 본문 표·영문초록·부록 상관표 사이 결과가 엇갈리는 지점(E/I 유의성, 유희적 사랑 상관 방향 등)이 있어 **수치 단위 인용은 피하고 방향성만** 사용.
- 사용자에게 이 분석이 **재미·자기이해용 참고이며 과학적 예측이 아님**을 (필요 시) 가볍게 고지.

## [부록] 용어 정의

### A. 사랑의 3요소 (Sternberg 사랑의 삼각형 이론)

| 요소 | 의미 |
|---|---|
| **친밀감 (Intimacy)** | 가깝고 연결되어 있다고 느끼는 따뜻한 정서적 유대. 관계의 '온도'. |
| **열정 (Passion)** | 신체적·성적 끌림과 낭만 등 사랑의 동기적·각성적 측면. 관계의 '불꽃'. |
| **헌신 (Commitment)** | 단기적으로는 사랑하기로 한 '결심', 장기적으로는 관계를 지키겠다는 '책임'. 관계의 '뿌리'. |

세 요소의 조합이 8가지 사랑 형태를 만듦(참고): 좋아함(친밀감), 도취적(열정), 공허한(헌신), 낭만적(친밀+열정), 우애적(친밀+헌신), 얼빠진(열정+헌신), 성숙한(셋 다), 비사랑(없음).

*참고 데이터:* 표본 전체 평균 친밀감(≈6.28) > 헌신(≈5.76) ≒ 열정(≈5.75) — 친밀감을 가장 높게, 열정을 가장 낮게 보고. 세 요소는 서로 +상관(특히 열정-헌신이 강함). *(수치는 방향성만 신뢰)*

### B. Lee의 6가지 사랑유형 (Love Styles)

| 한국어명 | 영문 | 한 줄 의미 |
|---|---|---|
| 좋은 친구 같은 사랑 (우애적) | Storge | 우정이 서서히 사랑으로. 안정적·점진적, 친밀·헌신 중시. |
| 낭만적 사랑 | Eros | 강한 신체적 끌림·열정 중심. 첫눈에 반하는 정열적 사랑. |
| 유희적 사랑 | Ludus | 사랑을 게임처럼 즐김. 얽매이지 않고 가볍게, 헌신 회피. |
| 논리적 사랑 | Pragma | 조건·실리를 따져 합리적으로 상대를 고르는 현실적 사랑. |
| 소유적 사랑 | Mania | 질투·집착이 강함. 감정 기복이 크고 애정을 끊임없이 확인. |
| 이타적 사랑 | Agape | 무조건적·자기희생적으로 베푸는 사랑. 보상을 바라지 않음. |

*참고 데이터:* 표본에서 평균이 가장 높은 유형은 '좋은 친구 같은 사랑(Storge)', 다음이 '이타적 사랑(Agape)'. 유희적(Ludus)·소유적(Mania)은 상대적으로 낮음.
